## 📖 About

Welcome to the digital playground of **Ashutosh Singare**, a Computer Science engineering student specializing in **Data Science, Applied AI, and Creative Web Architecture**. This repository contains the source code for an ultra-responsive, cinematic personal developer portfolio designed to push browser rendering boundaries.

Built specifically for **engineering recruiters**, **creative design agencies**, and **AI startups**, the portfolio merges mathematical visual effects with high-performance web engineering. It eliminates static layouts in favor of scroll-driven storytelling, interactive WebGL canvases, and dynamic state transitions.

Whether showcasing generative particle visualizers like **Neural Synth**, spatial 4D timelines like **Chronos Spatial**, or sub-second algorithmic trading interfaces like **Hyperion DEX**, this application demonstrates state-of-the-art frontend excellence with zero visual compromise.

## ✨ Features

- 🌌 **Cinematic 3D Scrollytelling**: Interactive **Lenis-powered scroll progression** driving dynamic Framer Motion transforms and WebGL particle nebulas.
- 🎨 **Glassmorphic Design System**: Custom HSL dark theme (`#121212`) enhanced with ambient gradient blur glows and micro-animated pill borders.
- ⚡ **60 FPS Rendering Pipeline**: Optimized canvas rendering and non-blocking DOM updates ensuring silky-smooth frame rates across all viewport sizes.
- 💼 **Dynamic Showcase Matrix**: Filterable project showcases featuring real-time production metrics, live external links, and tech stack tagging.
- 🎓 **Interactive Academic Timeline**: Clean chronological mapping of academic achievements from CBSE Secondary boards to B.Tech Data Science specialization at OIST.
- 📬 **Direct Commission Inquiry Flow**: Integrated mailto dispatcher and instant email clipboard utility for effortless client hiring and contact.
- 🛡️ **Strict Codebase Lockdown**: Adherence to AI engineering runtime principles protecting core UI components against unintended mutations.

## 🏗️ Architecture

The application relies on an asynchronous, reactive UI architecture decoupling canvas background rendering from interactive foreground DOM elements.

### System Flow Diagram (ASCII)

```text

 |                     Next.js 16 App Router Root                        |
 +-----------------------------------------------------------------------+
                                     |
         +---------------------------+---------------------------+
         |                                                       |
         v                                                       v
 +-------------------------------+       +-------------------------------+
 |  [ScrollyCanvas.tsx]          |       |  [Overlay.tsx]                |
 |  WebGL 3D Particle Background |       |  Scroll-driven Typography UI  |
 +-------------------------------+       +-------------------------------+
                                                         |
                                                         v
                                         +-------------------------------+

                                         |  (Projects, TechStack, Edu)   |
```

## 🎯 Usage

Here is how programmatic UI interactions are orchestrated within the portfolio codebase.

### 1. Animating Scroll-Linked Typography (`Overlay.tsx`)
We link Framer Motion values to window scroll progression to reveal text dynamically:

```tsx
import { motion, useTransform, MotionValue } from "framer-motion";

interface OverlayProps {
  scrollYProgress: MotionValue<number>;
}

export const HeroText: React.FC<OverlayProps> = ({ scrollYProgress }) => {
  // Fade out hero text between 0% and 22% scroll depth
  const opacity = useTransform(scrollYProgress, [0, 0.1, 0.22], [1, 1, 0]);
  const y = useTransform(scrollYProgress, [0, 0.22], [0, -100]);

  return (
    <motion.div style={{ opacity, y }} className="max-w-2xl text-left pr-6">
      <h1 className="text-7xl font-extralight text-white tracking-tight">
        Ashutosh <span className="font-normal bg-gradient-to-r from-white to-white/30 bg-clip-text text-transparent">Singare</span>
      </h1>
      <p className="text-2xl font-light text-white/60 mt-4">Creative Developer.</p>
    </motion.div>
  );
};
```

### 2. Triggering Instant Clipboard Copy (`Contact.tsx`)
Allow recruiters to copy your email with a single tap:

```tsx
const [copied, setCopied] = useState(false);
const emailAddress = "ashutoshsingarehere@gmail.com";

const handleCopy = () => {
  navigator.clipboard.writeText(emailAddress);
  setCopied(true);
  setTimeout(() => setCopied(false), 3000);
};
```

## 📡 API Reference

While this portfolio is primarily a static client-side Next.js app, it integrates with external web APIs and browser interfaces:

| Interface | Method / Protocol | Description | Payload |
| :--- | :--- | :--- | :--- |

## ☁️ Deployment

The application is optimized for instant deployment on edge networks like **Vercel**.

### Deploying to Vercel
1. Push your changes to your remote GitHub repository (`https://github.com/Ashutoshsingare/port1`).
2. Log in to [Vercel](https://vercel.com) and click **Add New Project**.
3. Import the `port1` repository.
4. Keep framework preset as **Next.js** (build command `next build`).
5. Click **Deploy**. Your portfolio will be live globally on an optimized CDN with automatic SSL.

## 🔧 Troubleshooting

| Problem | Likely Cause | Solution |
| :--- | :--- | :--- |
| **Animation jitter on mobile scroll** | Touch events conflicting with browser default overscroll behavior. | Ensure CSS includes `overscroll-behavior: none;` on body tag or check Lenis smooth scroll settings. |
| **WebGL canvas not rendering** | Hardware acceleration disabled in user browser. | Fallback CSS background gradient automatically activates in `ScrollyCanvas.tsx`. |
| **Port 3000 already in use** | Another local Node development server is active. | Launch with `npx next dev -p 3001` or terminate the background process. |
